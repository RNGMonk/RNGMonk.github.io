Interlaced Type
Before working on the effect you need to load the contents of the file interlace.psd in the clipboard. Click on the link below the page to download the file.

Type in the text and make sure its on a separate layer. The effect will have no result to plain black text.

Load a selection on the text and do not lose the selection.

With the selection still loaded, paste the contents of the clipboard into the selection by hitting CTRL + SHIFT + V. This should create the interlace effect but you can also change the layer mode from normal to softlight via the layers palette or simply reduce opacity to creat a more consistent intensity of the interlace based on the original texture or color of the text.
