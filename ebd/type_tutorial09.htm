Double Decker
Type in the text and make sure its on a separate layer. Select a color combination that works well together. My color combination for this tutorial is creamy yellow for the main font and dark red for the double deck font.

Make a selection appear on the text and then without losing the selection make a new layer below the current layer containing the text.

Without the selection still loaded, go to the new layer then fill the selection with the next color. You should now end up with two layers, one having the original text and the other with the expanded selection filled with the other selected color.


Apply the simple 3D effect on both the original text and the text created using the expanded selection. Displace the 3D effect using 1 pixel down and 1 pixel left.

Add a drop shadow or your own effects to complete the design.
