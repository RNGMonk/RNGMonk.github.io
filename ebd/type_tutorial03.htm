Glow
To create the glow effect you must first have the text on its own separate layer. Load a selection on the text and do not lose the selection.

Expand the current selection by 3 pixels. Make a new layer directly below the current layer and go to that layer.

With the selection still loaded, fill the current selection with the desired glow color and release the selection.

Check the preserve transparency box on the layers palette and make sure its unchecked. Apply BLUR>GUASSIAN BLUR and apply the desired strength of the glow.
