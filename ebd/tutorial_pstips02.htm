Photoshop Tips : Creating an outline using stroke



The STROKE option can be used to create border or paint along the paths with the desired color and thickness indicated in pixels. Here is a way to create a circle using stroke.

First you make the circular selection using the SHIFT key and the marquee tool.
fig. 1



Set the foreground to your desired border or stroke color then go to EDIT > STROKE . Use the dialog box to specify options like opacity, pixel thickness, etc (fig. 2).
