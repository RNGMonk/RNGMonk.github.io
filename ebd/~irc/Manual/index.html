FxZone Chat Server Help and Guide
 

New to IRC? IRC has a feel all to its own - it's not like the Forum, nor is it like instant messaging clients. The IRC world can be a very rough place for newbies, so soak in the following advice. Most certain read the following Dos and Donts. We also recommend the IRC Primer (at the bottom).
   
The Do's:
 
Please show consideration to everyone. Excessive personal attacking and baiting of others will result in being banned from the IRC server. Basically we ask you to follow the forum posting guidelines. If you are asked to take a conversation to private messaging, please do so. The beauty of IRC is that you can do this easily (by double-clicking on nick in most clients).
   
The Do Not's:
 

Excessive text color usage; excess bold/underline usage; caps locked talk (shouting); using or offering scripts; clones; unnecessary spam; unauthorised advertising; unwanted file sending; onjoin messages; begging for ops (will exclude you from consideration for several months); trolling. Again, we basically ask you to follow the forum posting guidelines.

Also, realize that many people chat on IRC while doing other things. Don't pop onto a channel, say "Hello" and then leave if no one responds in 30 seconds.
   
Getting setup with a local client:
 

The mainstream way of connecting to an IRC server is via a program called mIRC (for PC). To connect to the IRC server, run mIRC, then under Options, add a server. You can enter anything in the description box, and for the IRC server box, enter "fxzone.ultrairc.net port 6667". Click add, then click connect and you're... well, connected!

Other popular clients are BitchX (variety of platforms) and ircle for the Mac.

You can download one that we have setup for you here. You will need winzip to unzip the file to the directory of your choice.
