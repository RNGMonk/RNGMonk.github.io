Rock Surface
Select the layer containing the image or object where the texture will be applied.

Make a selection on the object or image and then make a new layer using the new layer button on the layers palette.

With the selection still loaded, select the newly formed layer and fill the selection with white.

Apply noise using Filters>Noise>Add Noise and make sure the monochromatic check box is checked. The amount of noise I used for this tutorial is 200 but you can experiment on your own settings.

Apply Filter>Stylize>Emboss on the object. 45 Degrees is the angle for the emboss, the height is 2 and the amount can range from 50-70 depending on how strong you want the roughness to be.

Change the current layer mode from normal to overlay via the layers palette to complete the application of the texture.
