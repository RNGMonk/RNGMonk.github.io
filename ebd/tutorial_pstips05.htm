Photoshop Tips : Optimizing web images in photoshop



Photoshop can also optimize images and convert them to GIFs. You can use the Indexed mode or export to GIF89Export to create to GIFs. The trick here is to be able to use the lowest number of colors or the lowest bit mode.

The image on the right is using a JPG format and has a file size of 4.7 k
Figure 01






I changed the image mode to IMAGE > MODE > INDEXED COLORS and used 7 bit colors and the file size is 2.52 k.
Figure 02






From the original JPG image, I changed the image mode to IMAGE > MODE > INDEXED COLORS and used 6 bit colors and the file size is 1.92 k.

Figure 03






From the original JPG image, I changed the image mode to IMAGE > MODE > INDEXED COLORS and used 7 bit colors and the file size is 1.52 k.

As you can see I was able to optimize and reduce the file size of the image to about 25%. This is similar to the concept they use for those image optimizing programs for GIFs with the only difference is that its automatically done when using these programs.
