3D Gradient Bar
Start by making a white 2d navigator bar with rounded edges and place it on a layer (layer 1).

Load a selection around the bar and do not lose the selection.

Use the gradient tool and apply gradient over the bar.

One color for the gradient should be dark and the other color should be light to create a very gradual gradient. The dark color should start from the bottom of the navbar.

Add the text and other effects(drop shadows, 3d effects, insets) to comlete the navigation bar or button.
