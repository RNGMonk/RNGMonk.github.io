Photoshop Tips : Memory optimization



fig. 1
Optimize your memory settings for photoshop® to be able to utilize other programs or maximize your memory usage if there is a shortage in RAM.

To change the memory settings, go to FILE > PREFERENCES > MEMORY AND IMAGE CACHE

If you have a lot of memory then it would be suggested you decrease the memory set for photoshop®, giving memory to other programs for better multitasking. If you have little memory you can maximize the memory set for photoshop®. This will kill the multitasking but it will improve the performance of the program.

75% is the amount of memory allocated to photoshop® by default and 64 mb is a recommended amount of memmory for photoshop®
