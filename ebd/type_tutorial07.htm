Metallic Shade
Type in the white text on a separate layer. Load a selection around the text and do not lose the selection.

With the selection still loaded, apply FILTERS>RENDER>LIGHTING EFFECTS (light position shown on the right). The settings I used for this tutorial :
Lightype : Spotlight
Intensity : 21
Focus : 100
Gloss : 0
Material : 100
Ambience : 8
Exposure : 0

Release the selection and apply FILTER > NOISE > ADD NOISE. Check the monochromic check box and apply light amount of noise. Add the additional effects to complete the design.
