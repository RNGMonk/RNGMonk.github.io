3D effect
The first steps for this tutorial is similar to creating a drop shadow. To start of, type in the text in its own separate layer. Duplicate the layer containing the text then go the lower copy and load a selection on the text. Do not lose the selection.

Fill the current selection with black then displace the text 3 pixels down and 3 pixels left then release the selection. This completes the step for this effect.

Drop shadows go well with this effect. The darker color of the original text may be used in the fill for the step above in order to create a more consistent 3d look.
