Emboss
To create the emboss effect, type in the text and make sure it appears on a separate layer. Make a duplicate of the layer with the text.

Go the the layer copy below and make a selection on the text. Then fill the current selection with white. After applying a white fill displace the text by 2 pixels up and 2 pixels to the right.

Release the selection then apply FILTER>BLUR>BLUR MORE and be sure that the preserve transparency box is unchecked before you apply the blur or it will take no effect on the text. Change the layer mode from normal to softlight of the current layer.

Make duplicate copy of the layer containing the white text. Go to one of the copy and invert the color of the contents by pressing CTRL + I. This should turn the color of the text from white to black.

Displace the contents of the new duplicate copy(containing black text) 4 pixels down and 4 pixels left
