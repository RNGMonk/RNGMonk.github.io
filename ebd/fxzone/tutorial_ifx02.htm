This is a follow up tutorial for my outer bevel tutorial. Using the contract option I was able to give the bevel effect a flat surface look but keeping the bevel effect at the edges.

First load a selection on the interface. Contract the selection. I used 6 pixels as the amount to contract the
selection but the amount will depend on the size of the actual interface. The bigger the interface the more you need to contact. Try to experiment with different settings.


With the selection still loaded, use the eyedropper tool to get the color of the center region of your interface.


Fill the selection using the color you just extracted from the center of the interface then you're done.
