Photoshop Tips : Monitor calibration


If your images come out too dark or too bright in other applications aside from photoshop, then you might need to calibrate your monitor.

You can accomplish this this by tiling the photoshop program and with a browser. I load a single image on both programs then I select FILE > COLOR SETTINGS then I click on the calibrate button. If the image on the browser looks darker then you need to reduce the gamma using the slider(fig. 1) to make it also darker or increase it if the image on the browser looks brighter.
