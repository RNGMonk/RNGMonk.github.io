Interlacing
The file interlace.zip contains a predesigned interlace lines that you can use which is downloadable at the bottom of this page. You will need this file for this effect.

To create the effect, Load the file interlace.psd to the clipboard then Load a selection on the image.

With the selection still loaded, press CTRL + SHIFT + V to paste the interlace lines into the current selection.

Change the layer mode of the interlace lines from normal to soft light via the layers palette.
