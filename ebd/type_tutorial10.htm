Diffused
Type in the text and make sure its on a separate layer. See if the preserve transparency box is unchecked in the layers pallete or you could be wondering why the next step wont work.


Apply Filter>Stylize>Diffuse to the current layer. Use Normal for the diffuse settings.

Make a selection on the text and expand the selection. I used 3 pixels for expanding the selection for this tutorial. Do not lose this selection.


With the selection still loaded, make a new layer below the current layer and go to that layer. Fill the current selection. The color for the fill should be either darker or lighter than the current color of the original text.


Apply a blur using Filters>Blur>Guassian Blur to the new layer with the filled selection. The amount of blur I used for this tutorial is 1.6 but you can experiment with your own settings. Reduce the opacity of the layer to a certain level(40%-70%). This is trial and error so use your best judgement for the amount of opacity the layer should have. The opacity level may even depend on the fill color you used.
