Double Gradient
Type in the text and make sure its on a separate layer. Select your foreground and background colors

Make a selection appear on the text and apply a vertical gradient. The Gradient option be linear. Hold the shift key and start the gradient from the bottom of the selection then drag it up to the upper tip of the selection. Do not lose the selection after this.

I used the darker color as my bottom color for this tutorial.

Expand the selection by a few pixels(I used 3 in this tutorial).


 Apply again a vertical gradient using the same colors but start from the upper end going down the bottom. Dont forget to hold the shift key while setting the start and end points of the gradients.

You can also swap the forgeround and background colors by pressing X and apply the gradient starting from the bottom up like what you did on step 2.

Apply the other effects to complete the image.
