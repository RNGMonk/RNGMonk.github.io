Texture fill
Before working with the actual text to be filled with a texture or image. Load first the image or texture that is going to be used as a fill in the clipboard. You can do this by opening the image then hit CTRL + A followed by CTRL + C. This will load the image or texture in the clipboard.

Working back on the text image, load a selection on the text and do not lose the selection.

With the selection still loaded, fill the selection with the image loaded in the clipboard by hitting CTRL + SHIFT + V which pastes into the current selection. A new layer containing the image fill with a mask will be created. You can move the position of the image inside the text by simply clicking on the layer thumbnail and using the move tool to move the contents in the desired position.
